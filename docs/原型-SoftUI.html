t检验：均值比较

t检验是比较两组数据均值的完美工具。它基于 t 分布，能够在 小样本 (n < 30) 且总体方差未知时，为你提供最可靠的结论。

# Python：独立样本 t检验调用
from scipy import stats

t, p = stats.ttest_ind(
  group_a, group_b, equal_var=False
)
print(f"P-value: {p:.4f}")
