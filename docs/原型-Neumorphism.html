t检验：均值比较

t检验是比较两组数据均值差异的核心工具。它基于 t 分布，专为 小样本 (n < 30) 且发散度未知的场景而设计。

// 独立样本检验 - Python
from scipy import stats

t, p = stats.ttest_ind(
  a, b, equal_var=False
)
print(f"P值: {p:.4f}")
