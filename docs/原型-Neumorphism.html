小节 1 / 2

t检验与小样本

t检验是比较均值差异的核心工具。它基于 t 分布，专为 小样本 (n < 30) 且发散度无从得知的场景而设计。

核心认知

理论上 t检验 要求数据必须符合正态分布。但在实际测算中，t检验实际上表现得非常“稳健”(Robust)。只要数据偏态不极其严重，直接使用常常也是足够安全的。

// 独立样本检验 - Python
from scipy import stats

t, p = stats.ttest_ind(a, b, equal_var=False)
print(f"P值: {p:.4f}")

B

小节 2 / 2

P值 的终极意义

运行完刚才的代码后，我们会得到一个关键数据：P值 (p-value)。这是假设检验中最核心的判决依据。

避坑提示

很多初学者会把 P值 误认为是“原假设为真的概率”。

其实不然！ P值的真正潜台词是：如果原假设本身是真的，那你观测到手头这批目前数据（甚至更夸张数据）的可能性有多低。

随堂练习

你运行代码得出 p=0.01。这严格意味着什么？

A
原假设有 1% 的可能是正确的
B
若原假设成立，出现刚才数据的概率只有 1%
