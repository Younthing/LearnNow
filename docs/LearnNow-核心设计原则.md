# LearnNow 设计原则

基于我们现有的新拟态 (Neumorphism) 原型，并深度融合当今面向成人终身学习的 EdTech 最佳实践（如 Duolingo 的激励机制、Memrise 的微学习感、高质量 SRS 系统等），LearnNow 的设计底层逻辑被提炼为以下 5 大核心准则。

这些准则是所有新功能开发、UI交互重构的最高“北极星指标”。

---

## 1. 微学习 (Micro-Learning)
**设计理念：**
成人学习者的专注时间高度碎片化，长篇大论极易引发弃学。我们需将复杂的领域知识拆解为极小认知颗粒度的“知识胶囊”，极大降低学习者的心理启动阻力。

**在原型的落地印证：**
- **Stories 式短平快交互**：彻底摒弃长文瀑布流，采用横向滑动吸附（Scroll-snap）。每一次滑动都像看社交动态一样轻松，每一页只承载 1 个明确的原子级知识点（如：“什么是 P值”）。
- **小步快走的成就感**：通过拆分内容，保证每次学习耗时不到 3 分钟，快速抵达游戏化的全屏通过结算页。


## 2. 少打扰 (Low Interruption)
**设计理念：**
用户在进入知识获取状态时，注意力极其宝贵。应用界面的 UI 应退化为“隐形辅助者”，绝不用枯燥繁琐的操作或者粗暴的打断弹窗去破坏心流（Flow）。

**在原型的落地印证：**
- **去处拦截的“防翻车”**：把原本做错题就不准翻页的强制阻断机制删除了。做题是一种主动验证，而不是拦路打劫。
- **上下文内嵌式反馈**：不再弹出巨大的全局 Dialog 告诉你“答错了”，而是将答案解析在选择按钮下方平滑展开并高亮，一切都在当前阅读视线内解决。


## 3. 高自由度 (High Freedom)
**设计理念：**
传统的学校式教育强调“必须按照我的章节顺序”，但这对于成年人是极度压抑的。我们必须尊重学习者的基础差异与知识跳跃性。

**在原型的落地印证：**
- **全局跳转穿梭雷达**：顶部的多段式进度条不仅是装饰，而是可点击的导航栏（Segmented Progress Bar）。允许用户随时跳过已懂模块，或者回退重温上一页。
- **把摇杆交给用户**：自由的手势横跨，无论是看知识、测验、还是跳关，应用随时随地回应你的操控，而不设立死板的“解锁条件”。


## 4. 沉浸式与强直觉 (Immersive & Intuitive)
**设计理念：**
学习本不是一件“舒服”的事情，我们必须通过最顶级的 UI/UX 体验，使其具备“解压游戏”般的舒适感和直觉物理反馈，以此抵消认知的痛苦。

**在原型的落地印证：**
- **拟真物理感 (Neumorphism)**：放弃冰冷的细线与生硬纯色。利用全局统一的 `neu-out` (高光外凸起) 与 `neu-in` (下凹内阴影)，营造出极其温润的“婴儿肌”与“橡皮泥”质感。
- **“学习 ➡️ 洞见 ➡️ 测验” 的防走马观花闭环**：在精美的单页中，完成知识传输、实战避坑（左侧亮色描边的 Caution Block）、再紧凑结合一个情境应用题（随堂确认），彻底把单页的知识吃干抹净。
- **顶级微动效赋能**：在用户答对时，满屏倾洒的五彩纸屑庆祝；在进入专属的结算页（Completion Screen）时，图标与结算面板如同瀑布般的阶梯浮现、连胜火焰的微微跳动呼吸……这些都为了最大化刺激多巴胺。


## 5. SRS 回路的透明显化 (Transparent Spaced-Repetition)
**设计理念：**
大多数学习应用只管“教”，不管“遗忘”。LearnNow 不仅负责间隔重复（Spaced Repetition System），还要把这个复习调度逻辑透明地展示给用户，建立系统的智能感与靠谱感。

**在原型的落地印证：**
- **认知的最终收口**：在最高光、用户心情最好的结算页上，不仅给 XP 经验值和连胜奖励，下方还会直接展示通过本次学习刚被生成的记忆胶囊卡片。
- **系统代工感**：用细腻的 UI 提示你“不用心智负担，智能调度系统已将考点编入你明日的艾宾浩斯复习池里”。
- **直接行动指引 (Frictionless CTAs)**：在这里可以直接利用情绪的高涨，把按钮引导为“继续下一课”或“直接前往卡片复习面板”。
