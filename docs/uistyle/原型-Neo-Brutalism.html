核心概念

t检验：均值比较

t检验是比较两组数据均值是否存在差异的大杀器。它基于 t 分布，专为 小样本 (n < 30) 且发散度未知的场景设计。

# Python：提取独立样本武器
from scipy import stats

t_stat, p_val = stats.ttest_ind(
    group_a, group_b, equal_var=False
)
print(f"P-VALUE ATTACK: {p_val:.4f}")
